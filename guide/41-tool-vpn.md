# 科学上网与 VPN

> 本文由 **卡网大全 [cardnav.xyz](https://cardnav.xyz)** 提供。

如果目标模型限制了你所在地区的直接访问，那么 VPN 或其他代理方式基本是绕不开的第一步。很多人遇到的问题，不是“有没有梯子”，而是“当前 IP 是否稳定、是否足够干净、是否和后续注册地区匹配”。

普通共享节点的优点是方便、便宜、现成就能用；缺点是同一个出口 IP 可能被很多人同时使用，一旦其中有人触发风控，其他人也可能受影响。

如果你更在意稳定性，可以把 VPN 需求拆成两类：

- 轻度使用：优先考虑成熟机场或代理服务
- 高要求使用：优先考虑自己的 VPS 节点

## Option: 云主机自建节点

常规的VPN由于是很多用户共同使用，所以他们的IP并不是很纯净。如果有人的账号在这些IP上被封号了，可能会影响到其他人的账号。如果你有更极致的追求，可以考虑自己购买云主机，自己搭建节点。

目前推荐用 vless + reality搭建。具体搭建方式，你可以在配置好云主机SSH后告诉 Codex 等AI工具，让AI直接帮你配置好。
如果你准备自己搭节点，可以继续看 [云服务器推荐](./42-tool-vps.md)。
