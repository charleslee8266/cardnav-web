# 云服务器推荐

> 本文由 **卡网大全 [cardnav.xyz](https://cardnav.xyz)** 提供。

如果你准备自建节点、运行反向代理程序，或者希望拥有更可控的网络环境，那么你通常需要一台自己的云服务器。对于个人或几个人的小规模使用场景，很多时候 `1C1G` 就已经够用了。

选择 VPS 时，主要看这几件事：

1. 线路质量和地区可用性
2. IP 相对纯净程度
3. 年付或月付成本
4. 服务商是否稳定、是否容易失联

一般来说，老牌服务商的优势不一定是最便宜，而是更稳、更容易排查问题。便宜套餐也不是不能用，但要接受网络质量和 IP 表现的不确定性更高。

这类 VPS 常见用途：

- 自建节点
- 部署自己的中转程序
- 作为更可控的海外出口环境

## Option：搬瓦工

搬瓦工是一个老牌云主机服务商，已经稳定运行了很多年。它的优势在于服务比较完善，响应速度快。并且提供CN2-GIA等线路。这些线路在中国大陆的访问速度足够快（如果你不在中国大陆地区，可以考虑其他选择）。另外，它们的IP地址相对比较纯净。
