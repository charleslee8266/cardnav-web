# Data and Content Policy

This repository open sources the CardNav software code. It does not grant a blanket license to copy CardNav's production data, merchant data, product data, search data, editorial content, screenshots, brand assets, or operational datasets.

## Software Code

Source code in this repository is licensed under the license in [LICENSE](./LICENSE), unless a file states otherwise.

## Production Data

The following are not licensed for bulk copying, resale, republishing, or operating a competing dataset without permission:

- Production database exports.
- Merchant, site, product, stock, price, and search datasets.
- User submissions, feedback, or moderation notes.
- Manually reviewed classification or curation results.
- Public ranking outputs, freshness data, and operational snapshots.

You may use the public website normally as an end user. Do not scrape CardNav at high volume, mirror the full dataset, or use CardNav as a substitute data feed without permission.

## Editorial Content and Assets

Guides, explanations, screenshots, logos, icons, brand assets, and public-facing copy are not automatically licensed for reuse just because the code is open source. Short excerpts with attribution are fine for commentary, reviews, and links back to the project.

## Attribution

If you publish a fork or derivative service, please clearly state that it is not the official CardNav service and link back to the official website:

- <https://cardnav.xyz>
